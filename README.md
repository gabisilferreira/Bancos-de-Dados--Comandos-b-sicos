# 📊 Exercícios de SQL: Gestão de Vendas

Este repositório contém a resolução de exercícios práticos de SQL focados na criação, manipulação e gerenciamento de um banco de dados de vendas.

## 🚀 Tecnologias Utilizadas
* **MySQL**
* **MySQL Workbench**

## 📂 Estrutura do Banco de Dados
O banco de dados `Exercicio` foi estruturado com três tabelas principais:
- **clientes**: Registro de clientes com nome, email, cidade e telefone.
- **produtos**: Catálogo de itens com preços e controle de estoque.
- **vendas**: Tabela de transações que relaciona clientes e produtos, com data e forma de pagamento.

## 🛠️ O que foi praticado:
- **DDL (Data Definition Language):** Criação de banco de dados, tabelas e alteração de estruturas (`CREATE`, `ALTER TABLE`).
- **DML (Data Manipulation Language):** Inserção de registros, atualização de dados e exclusão segura (`INSERT`, `UPDATE`, `DELETE`).
- **Resolução de Problemas:** Tratamento do *Error Code: 1175 (Safe Update Mode)* durante operações de exclusão.

## 📝 Como executar
1. Clone o repositório.
2. Importe o arquivo `exercicios.sql` no seu SGDB (MySQL).
3. Execute os comandos para visualizar os resultados.
